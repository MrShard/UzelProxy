Возможности расширения

  • Менеджер прокси
    Сохраняйте несколько прокси (HTTP/S и SOCKS5, включая TOR/WARP) и переключайтесь между ними. Активный прокси выбирается одним кликом.
  • Проверка соединения
    Кнопка «Проверить» для каждого прокси показывает, работает ли он, и выводит IP/страну выхода.
  • Готовый список исключений
    Открытый список доменов, которые открываются напрямую, минуя ваш прокси. Обновляется из репозитория в разделе «Управление».
  • Собственные списки сайтов
    Гибкая настройка того, какой трафик идёт через прокси:

    «Не проксировать домены из списка» — исключения: эти домены идут напрямую.
    «Проксировать только домены из списка» — через прокси идут только указанные домены, остальное напрямую.
    «Добавить домены для проксирования» — дополнительно проксировать выбранные домены.
    «Проксировать весь трафик» — весь трафик через прокси (режим VPN).
  • Отключение других прокси и VPN расширений
    Автоматическое отключение других прокси/VPN-расширений при конфликте за настройки прокси.

Все настройки доступны через меню расширения.
